StockFlow v19 Real Audit

هذه النسخة كاملة فعلية وتشمل:
- رفع المخزون
- رفع المبيعات
- رفع الشراء
- إعادة التوزيع
- المنتجات الراكدة
- تقرير الجرد لكل فرع
- KPI للأقسام
- Top 3 Problematic Departments
- Top 3 Performing Departments
- تصدير Excel
- تصدير Word عربي + إنجليزي

التشغيل:
- افتح run_app.bat
أو افتح index.html مباشرة.


تعديل v20:
- إزالة الاعتماد على أعمدة من/إلى في ملف الجرد.
- تحليل الجرد يعتمد فقط على Barcode + المنتج + الكمية + التكلفة + فئة المنتج.
- الكمية السالبة = عجز، والموجبة = أوفر.


تعديل إضافي:
- عند تصدير Excel الخاص بالتوزيع، يتم تضمين اسم المنتج داخل الملف.


تصحيح v21.1:
- الحفاظ على نفس واجهة الأبيض والبينك السابقة.
- إضافة Stock-out Analysis داخل نفس الواجهة بدون تغيير التصميم.


تصحيح v21.2:
- إصلاح زر تحليل الانقطاع.
- إضافة رسائل واضحة عند عدم تحميل الملفات.
- جعل قراءة أعمدة أول المدة والحركات أكثر مرونة.
- إظهار عدد السجلات بعد تحميل كل ملف.


تعديل v22:
- تطبيق ألوان اللوجو الأساسي #E61E8C مع تدرج #FF4DA6.
- إضافة اللوجو في واجهة التطبيق.
- إضافة زر تبديل اللغة عربي/إنجليزي.
- إضافة زر Dark Mode.
- قراءة الفئة Category أو الفئة من ملف المخزون.
- قراءة وحدة القياس Unit أو وحدة القياس من ملف المخزون.
- إضافة الفئة ووحدة القياس في جدول وتصدير إعادة التوزيع.
- Sweet and Sour لا يتم توزيعها بأقل من 5 قطع.
- المنتجات بوحدة KG/كجم/كيلو يتم توزيعها بحد أدنى 1.
- منع أن يكون نفس الفرع مصدر ومستقبل لنفس المنتج في نفس عملية التوزيع.


تعديل v22.2:
